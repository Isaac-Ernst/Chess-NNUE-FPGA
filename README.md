# Maximum-Efficiency-Chess-AI-with-Hardware-Integration
The goal of this project is to create a maximum efficiency chess AI using machine learning to build the evaluation function, low level programming with Verilog to increase speed on the specific hardware system drastically, and ultimately build the software in such a way as to push budget hardware to its maximum potential to create a grandmaster AI.

This project is intended to help improve my application to a masters program at a top 10 engineering school in the U.S. and as such this project will be extremely well documented.
